
## Getting Started

```bash
First, run the development server:
npm install
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
Then , run the scheduler server:
npm run scheduler  or start scheduler from UI (Dashboard )

```

Open [http://localhost:3010/systemup] with your browser to see the result.
